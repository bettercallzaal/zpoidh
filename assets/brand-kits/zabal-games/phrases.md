# ZABAL Gamez Phrases + Glossary

Mirror of the canonical phrases list from `github.com/ZAODEVZ/zabalgames/docs/brand-kit-2026-05-28.md`. Rotate the approved phrases sparingly. Banned phrases are hard NO across every public surface.

## Approved phrases (rotate sparingly)

- "Insert coin Mon Jun 1"
- "Pick your path. Insert coin."
- "Three doors. All real. Pick yours."
- "Watch. Learn. Build."
- "Free to join. Anyone welcome."
- "Use whatever harness you want."
- "Bring the build, we bring the team + distribution."
- "Build for a real community, not a weekend you forget."
- "Build event, not a video-game contest."
- "We learn it together."

## Banned phrases

| Banned | Why |
|---|---|
| "Hackathon" | We are a Build-A-Thon. 3 months, not a weekend. Mentor pairs. |
| "Pitch competition" | It's a build event. Working code matters, slides do not. |
| "Crypto-only" / "web3 native" (in PUBLIC copy) | Use "digital creators" framing instead |
| "Enter the metaverse" | Hard pass |
| Specific ZAO holder counts ("188 members") | Use "100+ members" |
| Specific past WaveWarZ stats | They are snapshots, point at the live URL |
| Emojis | Global ZAO rule, no exceptions |
| Em dashes (the long ones) | Use hyphens instead, global ZAO rule |
