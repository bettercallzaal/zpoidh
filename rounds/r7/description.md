zabal gamez (zabalgamez.com) is the build-a-thon platform - workshops, the portal, submissions, the season quest, the wins page. it has bugs. find a real one, fix it, get paid.

1. find a REAL bug on zabalgamez.com - something broken. a page that errors, a dead link, a mobile layout that breaks, a flow that dead-ends (signup, submission, quest, workshop signup).
2. fix it. open a public pull request with the fix. if you genuinely cannot PR, post a clear public writeup + patch showing exactly what to change and why.
3. prove it. a short screen recording or before/after screenshots showing the bug and the fix working.
4. submit the PR / writeup url on this poidh bounty page.

THE RUBRIC (more boxes = stronger)

impact
+ fixes something that actually blocks a user, not a typo nobody sees
+ fixes a mobile break - most builders are on their phones
+ fixes a broken core flow (submission, quest, signup, workshop signup)

craft
+ clean, minimal diff that fixes the bug without breaking anything else
+ includes a one-line note on how to verify the fix
+ handles the edge case, not just the happy path

extra
+ finds and fixes more than one bug in a single clean PR
+ leaves the code a little better than you found it where it helps

who wins
best fixes win and we merge them. this is not winner-take-all - multiple strong fixes can each get paid, because the whole point is making zabal gamez better. scope + payout per fix at the judge's call.
