## The idea

turns one recording into a wave of creator-made promo. Pair them: Unlock funds a POIDH
clipping bounty, clippers turn the ZABAL Gamez x POIDH fireside (the Unlock Protocol space
with Kenny, 2026-07-08) into short clips that sell Unlock. Every submission is someone
advertising Unlock for them - low lift, real distribution, and it shows Unlock the POIDH
model next to their existing Incented campaign.

This is the manual v1 of the recording -> clips -> POIDH bounty pipeline described in
ZAO OS V1 research doc 992.
