POID bounty deadline calendar

Proof of concept for zpoidh issue #6 - Kenny's idea from the 2026-07-08 ZABAL Gamez x POIDH fireside. POID bounties have no structured deadline field by design; this reads the stated deadline out of each bounty's free-text description instead. Built against BCZ's own R1-R4 bounties first - see scripts/build-bounty-calendar.py. Generalizing to arbitrary POID bounties is a follow-up, not done here.
