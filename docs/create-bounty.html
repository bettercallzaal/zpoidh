Create a POIDH bounty

Standalone tool - connects to any browser wallet (MetaMask, Coinbase Wallet, Rainbow, etc.), not gated to the Farcaster app. Calls the real POIDH contract on Base directly. Use this to spin up a clipping bounty (or any bounty) and to submit claims against one. See zpoidh for the full BCZ POIDH playbook this follows.

This is real money on Base mainnet. Creating a bounty locks ETH from your connected wallet into the POIDH contract. There is no test/sandbox mode here. Start small. Every action requires your wallet's own confirm - this page never holds or touches your keys.

Quick-fill: recurring series

Loads a bounty's template from its rounds/rN/description.md in this repo and fills in the version number, deadline, and winner-cast date automatically - version is derived live by counting how many bounties already exist on POIDH with this exact title, so re-running the same bounty next week needs no manual editing.
